# Simple App

A simple Node.js Express app with multiple routes and in-memory user data.

## Features

- `/` → Welcome message
- `/users` → Get all users (GET)
- `/users/:id` → Get single user by ID (GET)
- `/users` → Add new user (POST with JSON body)

## Project Setup

1. Install dependencies:
```bash
npm install
2. Run the app
npm start
3. Access in browser
http://localhost:3000
